已知缺口

致命

Embedding API 不可用

DeepSeek 不支持 embeddings 端点。语义轴的向量空间投影、Gram矩阵解耦、KNN检索全部空转。当前效果仅来自 state-delta 文本提示注入。需切换至支持 embedding 的模型（OpenAI text-embedding-3-small 或本地 BGE-Large-ZH）。

中等

d_style 恒为零

KNN 多因子排序中风格嵌入距离未训练/未加载，当前始终返回0。不影响语义准确性(因 d_fact 主导)，但降低了叙事风格匹配度。

中等

语义渲染未接入聊天气流

load_semantic_for_character() 和 _apply_semantic_renderer() 方法已定义，但 build_prompt() 尚未调用。语义渲染当前仅通过独立测试脚本调用。
